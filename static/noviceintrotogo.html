A Short Introduction to Go

Go (also known as Golang) is a statically typed, compiled programming language created by Google. Known for its simplicity, concurrency features, and efficient performance, Go aims to make developers more productive without sacrificing speed or clarity.

In the GOTTH stack, Go serves as the backbone for server-side logic, template rendering, and routing, providing the speed and reliability that power efficient web applications.

Now, Go is a pretty straightforward language, but it's got some tricky elements.

This tutorial assumes that you understand the basics (The := walrus, error handling, modules and imports) but if you don't, click one of the following links:

  • The official tour of Go. (My favorite) It's a set of tutorials with interactive elements, and it's extremely helpful for learning.


  • Go by example. This is a little bit less involved, but still teaches you everything with clear examples.


  • The Golang roadmap.sh page. Now, this is way more involved and rigorous compared to the other choices, and is aimed towards industry professionals. I'd reccomend doing the "learn the basics" section and stopping there.


  • The leetcode 75. Despite being a little less practical (Shots fired! Shots fired!) and despite it being much more involved than the other choices, you can still learn a lot about how go "Feels" and "Functions" through this. I reccomend doing 10 easy problems. If you're having trouble, search for Python/Rust/C answers, and then implement it using Go.
