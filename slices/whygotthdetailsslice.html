Hate working with .JS? Hate bloated programs that run slowly? Are you a new developer, looking for something that just "works"? Are you an experienced developer trying to dip into webdev? Come to GOTTH.

To put it simply: the larger javascript ecosystem is a mess. Dependencies upon dependencies, dual-code systems, hidden errors, performance issues - there's a lot to talk about.

To solve these issues, people create frameworks (and then more frameworks to fix issues of past frameworks), or they use tools like Django to handle full stack development.


Out of all of the full-stack frameworks, the GOTTH stack is one of the simpliest, easiest, and is customizable. If you want something that "just works" use GOTTH. Even though, you're using multiple tools, they all fit together neatly, and you can start cooking ASAP.
